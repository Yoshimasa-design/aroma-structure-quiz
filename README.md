# 香りから化学を学ぼう Ver.2.3

身近な香りを入口に、香気成分・構造式・化学を学ぶWeb教材です。

## 主な機能
- 香り → 成分名 → 構造式 → 官能基・分類の段階学習
- 香りクイズ、構造式クイズ
- お気に入り登録
- 復習モード（お気に入り、誤答、苦手、全成分）
- 分類色付き香気成分図鑑
- 学習記録（正答率、最終学習日、苦手成分）
- スマートフォン、タブレット、PC対応

## Ver.2.3の更新点
- お気に入り・間違えた問題・苦手な問題から選べる復習モードを追加
- 香り問題、構造式問題、混合問題を選択可能
- 化合物ごとに回答数、正誤、最終学習日時、回答モードを保存
- Ver.2.1の学習記録をVer.2.3形式へ自動移行
- 学習記録ページにお気に入り数、苦手成分数、最終学習日を追加
- 学習記録から復習モードへ移動できる導線を追加

## 重要な学習仕様
「香りから学ぶ」のSTEP 1正解画面では、構造式・分子式・官能基を表示しません。
構造式はSTEP 2の選択後に初めて表示されます。
