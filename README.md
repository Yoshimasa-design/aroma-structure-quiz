# 香りから化学を学ぼう Ver.2.2

身近な香りを入口に、香気成分・構造式・化学を学ぶWeb教材です。

## 主な機能
- 香り → 成分名 → 構造式 → 官能基・分類の段階学習
- 香りクイズ
- 構造式クイズ
- 分類色付き香気成分図鑑
- お気に入り登録と「お気に入りのみ」表示
- 学習記録（ブラウザのlocalStorage）
- スマートフォン、タブレット、PC対応

## Ver.2.2の更新点
- 図鑑の一覧カードと詳細画面にお気に入りボタンを追加
- 学習モードとクイズの解答画面にお気に入りボタンを追加
- 図鑑に「お気に入りのみ」フィルターを追加
- お気に入り保存処理を `js/storage.js` に分離
- Service Workerのキャッシュ名を更新

## 重要な学習仕様
「香りから学ぶ」のSTEP 1正解画面では、構造式・分子式・官能基を表示しません。
構造式はSTEP 2の選択後に初めて表示されます。
