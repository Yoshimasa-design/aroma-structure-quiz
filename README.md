# 香気成分クイズ

GitHub Pages向けの静的Webアプリです。

## 公開手順
1. GitHubで `aroma-structure-quiz` というPublicリポジトリを作成します。
2. このフォルダ内のファイルとフォルダをリポジトリ最上位へアップロードします。
3. `Settings` → `Pages` → `Deploy from a branch` を選びます。
4. Branchを `main`、フォルダを `/(root)` にして保存します。

## ローカル確認
このフォルダで `python3 -m http.server 8000` を実行し、`http://localhost:8000` を開きます。

## 注意
香りの表現と由来は教育用に代表例を簡略化しています。実際の香りは濃度、立体異性体、共存成分などで変化します。
